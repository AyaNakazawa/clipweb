
# clipweb

## Description
clipweb は、コードをブラウザ上で管理、共有できるウェブアプリケーションです。  
ECMAScript 6 をサポートしているブラウザで動作します。  
全ての通信にAjaxを利用するため、ページを再読込することはありません。  
一部の認証情報などの通信内容はSSL/TLSを使用しない場合でも暗号化、ハッシュ化されます。  
それぞれのコードは任意で暗号化することができます。
